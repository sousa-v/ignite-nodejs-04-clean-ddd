# Subdomínios

- Core: O que dá dinheiro
- Supporting: Dá suporta para o core funcionar
- Generic: Você precisa mas não são tão importantes

## Exemplos

### Core

- Compra
- Catálogo
- Pagamento
- Entrega
- Faturamento

### Supporting

- Estoque

### Generic

- Notificação ao cliente
- Promoções
- Chat
